# 🚀 WorkHive

### Where productivity meets collaboration.

WorkHive is a modern full-stack task and project management platform designed for both **individual productivity** and **team collaboration**. It helps users organize tasks, manage projects, track progress, and collaborate efficiently through an intuitive Kanban-style workflow.

## ✨ Features

### 🔐 Authentication & Security

* User Registration and Login
* JWT Authentication
* Protected Routes
* Password Hashing with bcryptjs

### 📋 Board Management

* Create Personal Boards
* Create Collaborative Boards
* Edit and Delete Boards
* Invite Team Members
* Manage Board Access

### 📌 Task Management

* Create, Update, Delete Tasks
* Task Priorities (Low, Medium, High, Critical)
* Due Dates
* Task Status Tracking
* Archive and Restore Tasks

### 🗂️ Kanban Workflow

* To Do
* In Progress
* Done

### 🎯 Task Assignment

* Assign Tasks to Team Members
* Reassign Tasks
* View Personal Assigned Tasks

### 💬 Collaboration

* Comments on Tasks
* Activity Tracking
* Team-Based Workspaces

### 🔍 Search & Filters

* Search Tasks
* Filter by Priority
* Filter by Status
* Filter by Assignee
* Filter by Due Date

### 📊 Analytics Dashboard

* Total Tasks
* Completed Tasks
* Pending Tasks
* Overdue Tasks
* Completion Statistics
* Productivity Charts

### 🔔 Notifications

* Task Assignment Notifications
* Comment Notifications
* Board Invitations
* Due Date Alerts

### ⚡ Real-Time Collaboration (Planned)

* Live Task Updates
* Live Comments
* Live Notifications
* Online Presence Tracking
